⚠️ **NOTA** ⚠️

Para poder correr el programa se necesita tener [SPIM](http://spimsimulator.sourceforge.net/) instalado.
En OSX, esto puede hacerse con [brew](https://formulae.brew.sh/formula/spim#default):
```
$ brew install spim
```
